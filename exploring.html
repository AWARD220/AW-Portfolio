Exploring QUT

November 21’
HTML, CSS

Exploring QUT was a University project that involved the task of creating a WCAG accessible, local Brisbane travel website aimed at users who were around 10 years old. Another interesting part of this project was it having to be completely responsive, allowing for full viewing at mobile and ipad viewports. For the website’s content I chose to go with a stylistic cartoon style and I chose to do locations around QUT for convenience. This project enforced the use of creative CSS use and a lot of accessible content such as skip link and breadcrumbs.

You can view this project at http://alexwportfolio.com/ExploringQUT/
