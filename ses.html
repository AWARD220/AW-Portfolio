Seminar Event System

November 21’
HTML, CSS, Bootstrap, SQL, Python

The Seminar Event System was a university project focusing on the back-end of a website and while I don’t usually pursue backend, it was extremely helpful in giving me an understanding of the ways both back and front-end communicate helping my UX and Web design experience overall. The object of the Website was to allow user with accounts to buy tickets, create events and manage events based on seminars. It also included a sign up/log in feature.

This was a collaborative project completed with Sam Geerling and Ella Riguet.
