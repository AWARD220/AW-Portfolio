catify.site

Try this project

NOTE: This project is currently waiting on approval from spotify before it can be accessed by the public. New sign in's will not work

catify.site is an app that makes cute cat characters based on your Spotify music. It's simple and lets you share a visual representation of your music taste - taking inspiration from other web application such as receiptify and How Bad is your Music Taste. The app takes the users top 3 artists and puts them into a category, like "Hip Hop" or "Pop." Then, it dresses up a cat with clothes that match those categories.

The web app requires the user to sign into spotify, selecting customization options to their liking and then sharing the image. catify.site also has distinct views for mobile and desktop and common outputs for consistant sharing across resolutions.

This web application was created in collaboration with Will Doyle.
